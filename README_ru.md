Этот проект учебный и служит небольшим конспектом моего изучения языка Java.

При решении каждой задачи проекта будут отрфбатываться новые знания и навыки.

* алгоритмы;
* структуры данных;
* конструкции языка;
* инструменты разработчика;
* DevOps;
* и т.п.
